# User Guide

This guide covers the user-facing parts of Synapseclaw: chat, web UI, Matrix, memory, automations, and skills. Several areas are still changing, so most pages stay short and focus on what is safe to rely on.

Skills are documented in depth because they are the most mature workflow feature. Start with [skills/quickstart.md](skills/quickstart.md) if you want the agent to remember and reuse a repeatable procedure.
