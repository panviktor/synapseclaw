# Web UI

The web UI is the browser control surface for working with the runtime. It is useful for checking status, interacting with the agent, and managing subsystems that have a web-backed flow.

The Skills page is the most complete area today: it can list memory-backed skills, create and update user skills, inspect candidates, diff/test/apply generated patches, and review health signals. Other web areas should be treated as evolving surfaces until their workflows settle.
