# What Is Synapseclaw?

Synapseclaw is a local agent runtime that can receive tasks through chat, web UI, and channels, then use tools, memory, and skills to complete them. It is designed to keep useful working knowledge without blindly expanding the model prompt.

The most mature user-facing workflow today is Skills. A skill turns a repeated procedure into a governed reusable capability that the agent can discover and load only when it is relevant.
